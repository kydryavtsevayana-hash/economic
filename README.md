# Экономический медиарадар

Статический образовательный сайт к курсовой работе на тему:
«Формирование критического мышления при анализе экономической информации в СМИ и социальных сетях на уроках экономики».

## Что входит в продукт

- Главная страница с описанием проблемы.
- Интерактивный анализатор экономических сообщений.
- Чек-лист критического анализа.
- Учебные кейсы для групповой работы.
- Мини-тест для самопроверки учащихся.
- Методический раздел для учителя экономики.
